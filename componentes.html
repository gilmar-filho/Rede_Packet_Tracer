📘 Componentes da Rede

Nesta página, você encontrará uma explicação sobre cada elemento presente na topologia de rede do nosso projeto.

🖥️ Hosts

São os dispositivos finais utilizados pelos usuários para acessar os serviços da rede. No nosso projeto, temos computadores e notebooks conectados à rede por cabo e por Wi-Fi. Eles acessam serviços como Web, FTP e DNS.

📡 Dispositivo Wi-Fi (Access Point)

Permite que dispositivos como notebooks se conectem à rede de forma sem fio. No nosso projeto, usamos um Access Point para conectar o Laptop3, funcionando como um host via Wi-Fi.

📶 Switch

O switch conecta todos os dispositivos da rede local, como os hosts e servidores, formando a infraestrutura da LAN. Ele funciona como um “distribuidor inteligente” de dados, enviando os pacotes para o destino correto.

🌐 Roteador

O roteador conecta duas redes diferentes. No nosso projeto, ele faz a ponte entre a rede dos hosts e a rede onde estão os servidores. Isso permite que os hosts se comuniquem com os servidores Web, DNS e FTP.

🧭 Servidor DNS

Responsável por traduzir nomes como ftp.redes.ufla em endereços IP como 192.168.1.20. Isso facilita o acesso aos serviços sem precisar decorar números de IP. Ele é essencial para o funcionamento por nome de domínio.

🌐 Servidor Web

Hospeda páginas HTML acessadas pelos hosts através do navegador. Ele usa o protocolo HTTP e serve como ponto principal de informação, com links e tutoriais para o uso da rede.

📁 Servidor FTP

Permite a transferência de arquivos entre hosts e o servidor. Os usuários podem fazer upload (enviar), download (receber), delete (excluir) e rename (renomear) arquivos, autenticando-se com login e senha. Utiliza o protocolo FTP (File Transfer Protocol).

🧱 Topologia Física da Rede

Nossa rede é dividida em dois blocos principais:

  • 🔹 Bloco 1: Hosts conectados ao Switch0, com um Access Point para conexão sem fio e acesso ao roteador.
  • 🔹 Bloco 2: Servidores conectados ao Switch1, que se comunica com o Switch0 através do roteador.

Essa divisão proporciona organização e controle do tráfego de dados entre os diferentes setores da rede.
